# Bitcoin Bay Skills

Claude Code skills used by [Bitcoin Bay Foundation](https://bitcoinbay.foundation), a 501(c)(3) nonprofit helping communities adopt Bitcoin through education, open-source tooling, and local meetups.

## Skills

| Skill | Description |
|-------|-------------|
| **open-meetup-bot** | Scaffolds and deploys a full-featured Bitcoin meetup website using the [open-meetup](https://github.com/AustinBitDevs/open-meetup) template. |
| **open-meetup-setup** | Step-by-step guide for configuring and launching an open-meetup site, covering integrations, content management, and production deployment. |

## Install

Add all skills:

```sh
npx skills bitcoinbayfoundation/skills
```

Add a specific skill:

```sh
npx skills bitcoinbayfoundation/skills --skill open-meetup-bot
```

## Contributing

If you're a Bitcoin Bay volunteer or community member with a useful skill to share, open a PR. Each skill lives in its own folder with a `SKILL.md` at minimum.
